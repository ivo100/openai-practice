# Hello World

This repository contains a minimal Python simple project managed with [Poetry](https://python-poetry.org/).

## Requirements

- Python 3.12 or later
- [Poetry](https://python-poetry.org/docs/#installation)

## Installation

Install dependencies by running:

```bash
poetry install
```

## Usage

Execute the simple script with:

```bash
poetry run simple
```

or
ave (source venv/bin/activate)

python simple/main.py

You can also run the module directly (in venv):

```bash
python -m simple.main
```
